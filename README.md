# iOS OSC Pencil
Send X-Y coordinates, RGB, and stroke using an Apple Pencil with an iPad.  

## How to build
Open OSC_Pencil.xcworkspace.    
Connect your ipad.    
Build it    

## Settings in iOS
Click the setting on the screen.     
Set the port 8800 and ip address of your computer.    
Press the “Set” button.

## Settings in Max/MSP
Just open the patch.
